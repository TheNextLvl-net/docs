# Editing Lines

Each hologram line has a variety of properties that can be customized.
These commands let you fine-tune the appearance and behavior of individual lines.

All edit commands follow the pattern:  
`/hologram line edit <hologram> <line> <property> <value>`

## Text Editing

### Alignment

Sets the text alignment of a line.  
Usage: `/hologram line edit <hologram> <line> alignment <alignment>`

**Examples:**
```
<gray># Center-align text</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> alignment <green>CENTER</green>

<gray># Left-align text</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> alignment <green>LEFT</green>

<gray># Right-align text</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> alignment <green>RIGHT</green>
```

### Append Text

Appends text to the end of a line.  
Usage: `/hologram line edit <hologram> <line> append <text>`

**Examples:**
```
<gray># Append text to a line</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> append <green> - Updated!</green>

<gray># Add a suffix</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>2</yellow> append <green> [NEW]</green>
```

<Callout type="info">
  Text lines support line breaks using the `<br>` and `<newline>` tags.
</Callout>

## Images in Text Lines

Text lines support rendering images using the `<image>` tag.  
The image is resolved from a file path or URL and rendered as colored block characters.

Usage: `<image:'source'>`  or `<image:'source':'height'>`

The `source` can be a local file path or a URL (`http://` / `https://`).  
The `height` parameter controls how many lines tall the image is rendered (defaults to `8`).

Images are cached for 30 minutes after first load.

**Examples:**
```
<gray># Display an image from a URL</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> set text <green><image:'https://example.com/logo.png'></green>

<gray># Display an image from a local file</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> set text <green><image:'/path/to/image.png'></green>

<gray># Display an image with a custom height of 16 lines</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> set text <green><image:'https://example.com/logo.png':'16'></green>
```

<Callout type="info">
  The image tag can be combined with other text and MiniMessage tags in the same line.
</Callout>

### Transformation

Sets how an item line is displayed, changing the way the item is rendered.  
This only applies to item lines.  
Usage: `/hologram line edit <hologram> <line> transformation [<transformation>]`

Available transformations:

| Transformation | Description |
|---|---|
| `NONE` | No transformation applied |
| `FIXED` | Rendered as a fixed item (default) |
| `GROUND` | Rendered like an item dropped on the ground |
| `GUI` | Rendered like an item in a GUI/inventory slot |
| `HEAD` | Rendered like an item worn on the head |
| `FIRSTPERSON_LEFTHAND` | Rendered as held in first-person left hand |
| `FIRSTPERSON_RIGHTHAND` | Rendered as held in first-person right hand |
| `THIRDPERSON_LEFTHAND` | Rendered as held in third-person left hand |
| `THIRDPERSON_RIGHTHAND` | Rendered as held in third-person right hand |

**Examples:**
```
<gray># Render like an item in a GUI</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> transformation <green>gui</green>

<gray># Render like a drop on the ground</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> transformation <green>ground</green>

<gray># Render like an item on the head</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> transformation <green>head</green>

<gray># View current transformation</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> transformation
```

### Scale

Sets the scale of a line.  
Usage: `/hologram line edit <hologram> <line> scale <x> [<y> <z>]`

Providing a single value scales all axes uniformly.

**Examples:**
```
<gray># Double the size uniformly</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> scale <green>2</green>

<gray># Scale each axis individually</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> scale <green>2</green> <green>1</green> <green>0.5</green>

<gray># Half the size uniformly</gray>
/hologram line edit <aqua>my-hologram</aqua> <yellow>1</yellow> scale <green>0.5</green>
```

### Offset

Usage: `/hologram line edit <hologram> <line> offset <x> <y> <z>`

**Examples:**
