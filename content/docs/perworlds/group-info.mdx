---
title: Group information
description: How to view group information and list all existing groups.
---

## List all groups

To view all existing groups, use the command `/world group list`.

This will print a list of all groups that have been created.

## View group details

To view all worlds assigned to a specific group, use the command `/world group info [<group>]`.

This will print all the worlds that are currently assigned to the specified group.
