# Plugin Comparison

This page provides an objective comparison between **PerWorlds** and other per-world inventory plugins for Minecraft servers.

<Callout type="info" title="Disclaimer">
  This comparison was researched to the best of our ability.  
  If you find any inaccuracies, please [open an issue](https://github.com/TheNextLvl-net/docs/issues/new) so we can correct them.
</Callout>

|--------------------------|----------------------------------------------------|--------------------------------------------------------------------------------|------------------------------------------------------|

| **Folia Support**        | ❌                                                  | ❌                                                                              | ❌                                                    |

| **World Groups**         | ✅                                                  | ✅                                                                              | ✅                                                    |
| **Open Source**          | ✅ GPL-3.0                                          | ✅ BSD-3                                                                        | ✅ MIT                                                |
