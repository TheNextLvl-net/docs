---
title: Skins
description: Manage skins for your player characters.
---

To change the character's skin, use the command '/character skin <layer/reset/set> <CHARACTER_NAME> <file/user/URL> [<USER/PATH/URL>]'

## Example

To change a character's skin to the skin of "Notch", with the characters name being "EXAMPLE" you can use the following command:

```
/character skin set EXAMPLE user Notch
```
