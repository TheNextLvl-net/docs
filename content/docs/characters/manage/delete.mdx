---
title: Delete a character
description: How to delete a character.
---

To delete a character, use the command `/character delete <character>`.

<Callout type="error">

Deleting a character is permanent and cannot be undone.  
 Make sure to back up any important data before proceeding.

</Callout>

## Example [!toc]

```
/character delete <aqua>Steve</aqua>
```
