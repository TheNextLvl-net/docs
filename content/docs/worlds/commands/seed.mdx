# World Seed

The seed command displays the seed of a specific world.

**Usage:** `/seed [<world>]`

**Permission:** `worlds.command.seed`

<Callout type="info">
  See the [Permissions page](/docs/worlds/permissions) for all command permissions.
</Callout>

<Callout type="info">
  This command is also available as a standalone command: `/seed`
</Callout>

The `<world>` argument is optional. If not specified, the command will display the seed of the world you're currently in.

<Callout type="tip">
  The seed in the command output is clickable! Click on it to copy the seed to your clipboard for easy sharing or reuse.
</Callout>

## Examples

View the seed of your current world:
```
/seed
```

View the seed of a specific world:
```
/seed <aqua>worlds:survival</aqua>
```
