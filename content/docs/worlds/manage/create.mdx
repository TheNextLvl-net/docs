title: Create a new world
---

You can create new worlds with the `/world create <key>` command.   
There are three options `generator`, `type`, and `preset`.

To access the `/world create` command, the permission `worlds.command.create` is required

## Generator

To generate a new world using a custom generator, use the command   
`/world create <key> generator <generator> [<dimension>] [<structures>] [<seed>]`   
The generator argument takes the name of the plugin that provides the custom chunk generator or biome provider.

For example:
```
/world create city:plots generator PlotSquared
```

## Type

World types denote the type of world you want to create.   
The following types are available:
- `minecraft:noise` - [The default world generation.](https://minecraft.wiki/w/Dimension_definition#noise)
- `minecraft:flat` - [A superflat world.](https://minecraft.wiki/w/Dimension_definition#flat)
- `minecraft:large_biomes` - [All biomes are larger.](https://minecraft.wiki/w/Large_biomes)
- `minecraft:amplified` - [Generates amplified terrain.](https://minecraft.wiki/w/Amplified)
- `minecraft:fixed` - [Uses one specified biome everywhere.](https://minecraft.wiki/w/Dimension_definition#fixed)
- `minecraft:debug` - [Debug all block states.](https://minecraft.wiki/w/Dimension_definition#debug)


## Preset

World presets are a way to create worlds with specific settings.
The following presets are available:
- `classic-flat` - [A classic flat world, no Strongholds unlike in default superflat.](https://minecraft.wiki/w/Superflat#Default_presets)
- `tunnelers-dream` - [Mineshafts, Strongholds, and Dungeons.](https://minecraft.wiki/w/Superflat#Default_presets)
- `water-world` - [A world with a lot of water.](https://minecraft.wiki/w/Superflat#Default_presets)
- `overworld` - [A flat world with ores, trees, and other features.](https://minecraft.wiki/w/Superflat#Default_presets)
- `snowy-kingdom` - [A snowy flat world.](https://minecraft.wiki/w/Superflat#Default_presets)
- `bottomless-pit` - [Cobblestone instead of bedrock.](https://minecraft.wiki/w/Superflat#Default_presets)
- `desert` - [A desert flat world.](https://minecraft.wiki/w/Superflat#Default_presets)
- `redstone-ready` - [A redstone ready world.](https://minecraft.wiki/w/Superflat#Default_presets)
- `the-void` - [A void world.](https://minecraft.wiki/w/The_Void)
