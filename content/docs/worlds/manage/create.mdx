You can create new worlds with the `/world create <key>` command.  
There are three options `generator`, `type`, and `preset`.

<Callout type="info" title="Frequesntly asked questions">
  [What does \<key\> stand for, in command arguments?](/docs/worlds/faq#key-argument)   
  [Why keys instead of world names?](/docs/worlds/faq#why-keys)
</Callout>

## Dimensions

There are following dimensions types:

- `minecraft:overworld` - [The primary dimension.](https://minecraft.wiki/w/Overworld)
- `minecraft:the_nether` - [The Nether dimension.](https://minecraft.wiki/w/The_Nether)
- `minecraft:the_end` - [The End dimension.](https://minecraft.wiki/w/The_End)

## Generator

To generate a new world using a custom generator, use the command  
`/world create <key> generator <generator> [<dimension>] [<structures>] [<seed>]`  
The generator argument takes the name of the plugin that provides the custom chunk generator or biome provider.

### Example [!toc]

PlotSquared generation example:

```
/world create city:plots generator PlotSquared
```

## Type

To generate a new world using a predefined world type, use the command  
`/world create <key> type <type> [<dimension>] [<structures>] [<seed>]`

World types denote the type of world you want to create.  
The following types are available:

- `minecraft:noise` - [The default world generation.](https://minecraft.wiki/w/Dimension_definition#noise)
- `minecraft:flat` - [A superflat world.](https://minecraft.wiki/w/Dimension_definition#flat)
- `minecraft:large_biomes` - [All biomes are larger.](https://minecraft.wiki/w/Large_biomes)
- `minecraft:amplified` - [Generates amplified terrain.](https://minecraft.wiki/w/Amplified)
- `minecraft:fixed` - [Uses one specified biome everywhere.](https://minecraft.wiki/w/Dimension_definition#fixed)
- `minecraft:debug` - [Debug all block states.](https://minecraft.wiki/w/Dimension_definition#debug)

### Example [!toc]

Amplified world example:

```
/world create amplified_test_world type minecraft:amplified
```

## Preset

To generate a new world using an existing preset, use the command  
`/world create <key> preset <preset> [<dimension>] [<structures>] [<seed>]`

World presets are a way to create worlds with specific settings.  
Here you can find a list of [available presets](/docs/worlds/presets#available-presets).

Want to learn how to create your own presets?  
Check out the [Creating a world preset](/docs/worlds/presets) documentation.

### Example [!toc]

Void world example:

```
/world create my_void_world preset the-void
```
