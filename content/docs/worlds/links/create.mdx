---
title: Create world links
description: How to create links between worlds.
---

To create a world link you need to have at least two worlds created.  
You can create a link by using the command `/world link create <source> <destination>`.

The `<source>` world has to be of the dimension type `Overworld`.  
The `<destination>` world must be one of `Nether` or `End`.

## Example [!toc]

Linking an **Overworld** to a **Nether** world:

```
/world link create farmworld:normal farmworld:nether
```

Linking an **Overworld** to an **End** world:

```
/world link create farmworld:normal farmworld:the_end
```
