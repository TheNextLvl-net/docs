# Plugin Comparison

This page provides an objective comparison between **goPaintAdvanced**, **BetterGoPaint**, and the original **goPaint**.

<Callout type="info" title="Disclaimer">
  This comparison was researched to the best of our ability.  
  If you find any inaccuracies, please [open an issue](https://github.com/TheNextLvl-net/docs/issues/new) so we can correct them.
</Callout>

## Overview

| Feature                 | [goPaintAdvanced](https://modrinth.com/project/a2wQ6jIv) | [BetterGoPaint](https://github.com/OneLiteFeatherNET/BetterGoPaint) | [goPaint](https://github.com/Arcaniax-Development/goPaint_1.14) |
|-------------------------|----------------------------------------------------------|---------------------------------------------------------------------|-----------------------------------------------------------------|
| **Server Software**     | Paper, Folia*                                            | Bukkit, Paper                                                       | Bukkit, Spigot, Paper                                           |
| **Folia Support**       | ⚠️ Requires unsupported FAWE version                     | ❌                                                                   | ❌                                                               |
| **MC Versions**         | 1.20.6 – latest                                          | 1.20 – 1.21                                                         | 1.8 – 1.17                                                      |
| **Open Source**         | ✅ GPL-3.0                                                | ✅ GPL-3.0                                                           | ✅ GPL-3.0                                                       |
| **Actively Maintained** | ✅                                                        | ❌                                                                   | ❌                                                               |
| **FAWE Required**       | ✅                                                        | ✅                                                                   | ✅                                                               |
| **Developer API**       | ✅                                                        | ❌                                                                   | ❌                                                               |
